# TxStepCounterSrv
リアルタイム歩数計送信アプリ（**TkPeerStepCounter**）のデータ受信プログラム

歩数計情報を受信したプログラムをKey-Upのキーボード疑似操作を行います。

Google　Mapのストリートビューにフォーカスし、Key-Upを行うとストリートビューのマップを移動することができます。
ステッパーなどで運動しながら、ストリートビューのマップを移動を楽しむことができます。
